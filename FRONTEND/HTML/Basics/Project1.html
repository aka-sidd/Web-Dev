Data Structures and Algorithms

Data Structures and Algorithms(DSA) is a fundamental subject in Computer Science. It is about how to organize data effeciently and retrive easily and how to write procedures to solve problems with that data(Algorithms)

Arrays

An array is a collection of items of the same data type stored at contiguous memory locations.It is the simplest data structure that can be used to store a collection of data, but it has some limitations.

Linked List

A linkedlist is a data structure , in which the elements are not stored in a contiguous memory location.The element are linked list are linked using pointer, where each element (a node) point to the next node

Tree

A tree is a non linear data structure that consist of nodes connected by edges. Each tree has a root from which other node connected

Binary Tree

A binary tree is a specific type of data Structure that has only two children for each node referred as a left child or right child and there are many types like Strict Binary tree Complete Binary tree, Full Binary tree and Perfect Binary tree

Binary Search tree

A Binary Search tree is a special type of Binary tree where the nodes are ordered.For any given node,the left subtree contains only nodes with keys less than the node's key, and the right subtree contains only nodes with keys greater than the node's key.

Key Properties of BST

The key properties of a Binary Search Tree are:

  • For any given node, all nodes in its left subtree have keys less than the node's key.
  • For any given node, all nodes in its right subtree have keys greater than the node's key.
  • Both left and right subtrees are also binary search trees.
